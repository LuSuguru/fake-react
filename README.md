# fake-react

基于官方 React 16.8.6 的源码并使用 TypeScript 仿写的 React，便于学习和掌握其原理

- [x] virtual DOM
- [x] DOM render
- [x] Fiber Reconciler
- [x] class component and setState
- [x] 事件机制
- [x] functoin component and hook
- [x] diff 
- [x] 一些常见的api，如memo,fragment等

## 使用
``` sh
  git clone https://github.com/LuSuguru/fake-react.git

  yarn watch
  yarn build

  // test
  cd example
  yarn start
````
